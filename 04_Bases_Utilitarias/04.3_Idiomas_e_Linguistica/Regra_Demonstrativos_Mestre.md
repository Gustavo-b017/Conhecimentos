# Pronomes Demonstrativos (This, That, These, Those)

#### [[This]] 
* Usado para indicar que tem algo perto
#### [[That]]
* Usado para indicar que tem algo distante


---

#### [[These]] 
* Usado para indicar que tem algumas coisas próximas

#### [[Those]] 
* Usado para indicar que tem algumas coisas distantes

* Usado para indicar que tem algo proximo
* Exemplo:
	* ( + )
		* This is an apple ( isso é uma maça )
	* ( - )
		* This isn't a car ( isso não é um carro )
		* This is not a car
	* ( ? )
		* Is this a car ?

Usado para indicar que tem algo distante
* Exemplo:
	* ( + )
		* That is an apple ( aquilo é uma maça )
	* ( - )
		* That isn't a cat ( aquilo não é um gato )
		* That is not a cat
	* ( ? )
		* Is that a cat ?

 para indicar que tem algumas coisas próximas
* Exemplo:
	* ( + )
		* These are oranges ( essas são laranjas )
	* ( - )
		* These aren't pens ( essas coisas não são lapis )
		* These are not pens
	* ( ? )
		* Are these a cars ?

* Usado para indicar que tem algumas coisas distantes
* Exemplo:
	* ( + )
		* Those are oranges ( aquilas coisas sao laranjas )
	* ( - )
		* Those aren't pens ( aquilas coisas nao sao lapis )
		* Those are not pens
	* ( ? )
		* Is those cats ?
