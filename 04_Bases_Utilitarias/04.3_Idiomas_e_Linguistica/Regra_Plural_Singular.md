| Singular                                                                                          | Plural                                                                        | exemples                                                                                              |
| ------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| Nouns that end in a **consonant + ' y '** <br><br>substantivos que terminam com consoante e ' y ' | Change ' y ' to ' i ' and + ' es '<br><br>tira o ' y ', coloca ' i ' e ' es ' | baby - babies<br><br>city - cities<br><br>strawberry - strawberries<br><br>secretary - secretaries    |
| Nouns that end in a **vowel + ' y '** <br><br>substantivos que terminam com vogais e ' y '        | Add + ' s '<br><br>adiciona o ' s '                                           | toy - toys<br><br>key - keys<br><br>monkey - monkeys<br><br>boy - boys                                |
| Nouns that end with a **s/ss/sh/ch/x/o** <br><br>substantivos que terminam com s/ss/sh/ch/x/o     | Add + ' es '<br><br>adiciona o ' es '                                         | potato - potatoes<br>bus - buses<br>kiss - kisses<br>tax - taxies<br>watch - watches<br>dish - dishes |
| I am<br>You are<br>He<br>She<br>It                                                                | We are<br>You are<br>They are                                                 | I am OK - We are OK<br><br>You are man - You are man<br><br>He is nice - They are nice                |
| This / That                                                                                       | These / Those                                                                 | This is a car - These are cars<br><br>That is a pen - Those are pens                                  |
| an / an                                                                                           | no plural                                                                     | It is a car. They are cats                                                                            |
| nouns that irregular<br><br>substantivos que são irregulares                                      | change the vowel or<br>change the word                                        | man - men<br>woman - women<br>foot - feet<br>child - children<br>mouse - mice<br>person - people      |
| Unchanging nouns<br><br>substantivos imutaveis                                                    | singular and  plural are the same<br><br>singular ou plural fica o mesmo      | sheep<br>deer<br>fish                                                                                 |
| ------                                                                                            | Nouns that have only a plural form<br><br>substantivos que so tem no plural   | glasses<br>scissors<br>binoculars<br>jeans<br>pajamas<br>shorts<br>sunglasses                         |
| Nouns that have only a singular form<br><br>substantivos que so tem no singular                   | -------                                                                       | information<br>baggage<br>grass<br>advice<br>knowledge<br>news<br>furniture                           |

#### Exceptions:

* studio - studios
* piano - pianos
* kangaroo - kangaroos
* zoo - zoos
* radio - radios
* spoon - spoons

#### **DO NOT** use plural forms with adjectives ou colors
* Ex: 
	* big houses
	* blue pens
* Não se coloca adjetivos ou cores no plural
